# NPS_peak_sigma
Jupyter notebook for calculating the peak and sigma of a 1D NPS.
Also it fits the parametrized NPS using the formula's with 2 and 5 parameters.

Software license: MIT.

## Package dependencies
- numpy
- scipy
- matplotlib
